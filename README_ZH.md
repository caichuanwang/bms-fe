## 安装使用

- 获取项目代码

```ssh
 git clone https://github.com/caichuanwang/bms-fe.git
```

- 安装依赖

```
 cd bms-fe
```

```
 yarn install
```

- 运行

```
 yarn start
```

- 打包

```
 yarn build
```
